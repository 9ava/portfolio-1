# 구아바약국 웹사이트

포트폴리오용으로 제작한 가상의 약국 홍보 웹페이지입니다.

## 데모

https://9ava.github.io/portfolio-1/

## 주요 기능

- 반응형 디자인 (모바일/태블릿/데스크톱)
- 히어로 섹션 이미지 슬라이더
- 구글 지도 연동
- 스무스 스크롤 네비게이션

## 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript

## 페이지 구성

- 메인 비주얼 (이미지 슬라이더)
- 약국 소개
- 핵심 가치 (신뢰, 정성, 건강)
- 서비스 안내
- 이용 안내 (영업시간, 연락처)
- 오시는 길 (구글 지도)

## 참고

본 웹사이트는 포트폴리오 목적으로 제작되었으며, 실제 약국과는 관련이 없습니다.
